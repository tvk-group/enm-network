Carteras adicionales recomendadas

Para un lanzamiento de token completo, considere crear estas carteras adicionales cuando esté listo:

Presale Escrow / Sale

Cartera de venta dedicada, separada de Treasury si desea aislamiento operativo (opcional — Treasury puede cumplir este rol).

Vesting & Locks

Para calendarios de vesting de equipo, fundadores e inversores cuando se despliegue el vesting on-chain.

Liquidity

Para provisión de liquidez DEX cuando se anuncie oficialmente.

Community & Rewards

Para programas piloto, incentivos comunitarios y asignaciones a socios.
