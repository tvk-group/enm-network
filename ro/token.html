Despre ENM

EnergieMIND Token (ENM) este tokenul utilitar Ethereum ERC-20 pentru ecosistemul EnergieMIND. Este conceput pentru coordonarea viitoare legată de inteligența energetică, participarea la utilitatea căldurii miniere, programe pilot, stimulente pentru infrastructura energetică și alinierea la mainnetul EnteleKRON.

ENM este separat de site-ul proiectului EnergieMIND de pe energiemind.com, care acoperă tehnologia și infrastructura. Acest site documentează doar stratul de token.
